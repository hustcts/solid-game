根据中国 K12 数学课程标准，立体几何主要内容包括：

| 学段 | 年级 | 核心内容 |
|------|------|----------|
| **小学** | 1-2 年级 | 认识长方体、正方体、圆柱、球 |
| **小学** | 3-4 年级 | 观察物体（从不同方向看）、简单展开图 |
| **小学** | 5-6 年级 | 表面积和体积计算、单位换算 |
| **初中** | 7 年级 | 点线面关系、基本几何体性质 |
| **初中** | 8 年级 | 三视图、展开图、截面 |
| **初中** | 9 年级 | 圆锥圆柱计算、空间想象 |
| **高中** | 必修二 | 空间几何体、位置关系、体积表面积综合 |

## 🎯 游戏模式设计（6 大模式）

### 模式 1: 🔍 视图配对（三视图训练）

**学习目标**: 培养空间想象力，理解从不同方向观察物体的概念

**玩法**:
- 屏幕中央显示一个 3D 几何体（可旋转观察）
- 下方显示 4 组三视图（主视图、俯视图、左视图）
- 玩家选择正确的一组三视图
- 答对后，3D 几何体会动画展示三个视角的转换

**难度分级**:
- 🟢 简单：单一几何体（立方体、球体等）
- 🟡 中等：2-3 个几何体组合
- 🔴 困难：复杂组合体，需要空间推理

**对应课程**: 小学 3-4 年级、初中 8 年级

---

### 模式 2: 📐 展开图配对

**学习目标**: 理解立体图形与平面图形的转换关系

**玩法**:
- 显示一个 3D 几何体（如正方体、圆柱、圆锥）
- 下方显示 4 个平面展开图选项
- 玩家选择正确的展开图
- 答对后，播放 3D→2D 的展开动画

**几何体类型**:
- 正方体（11 种展开图）
- 长方体
- 圆柱（侧面展开是长方形）
- 圆锥（侧面展开是扇形）
- 棱柱、棱锥

**对应课程**: 小学 4-5 年级、初中 8 年级

---

### 模式 3: 🔪 截面猜猜看

**学习目标**: 理解平面截立体图形得到的截面形状

**玩法**:
- 显示一个 3D 几何体（如立方体、圆柱、圆锥）
- 显示一个平面以某种角度切割几何体的动画
- 玩家预测截面的形状（三角形、四边形、圆形、椭圆等）
- 答对后，展示完整的截面形成过程

**典型题目**:
- 用平面截正方体，可能得到什么形状？
- 平行于底面截圆锥，得到什么形状？
- 斜着截圆柱，得到什么形状？

**对应课程**: 初中 8-9 年级

---

### 模式 4: 🧮 体积/表面积计算挑战

**学习目标**: 掌握立体图形的体积和表面积公式

**玩法**:
- 显示一个 3D 几何体，标注尺寸（长、宽、高、半径等）
- 给出问题："求体积"或"求表面积"
- 玩家从 4 个选项中选择正确答案
- 答对后，展示计算过程和公式

**公式覆盖**:
- 长方体：V = lwh, S = 2(lw + lh + wh)
- 正方体：V = a³, S = 6a²
- 圆柱：V = πr²h, S = 2πr² + 2πrh
- 圆锥：V = (1/3)πr²h
- 球体：V = (4/3)πr³, S = 4πr²

**难度分级**:
- 🟢 简单：直接代入公式
- 🟡 中等：需要单位换算
- 🔴 困难：组合体、缺角体

**对应课程**: 小学 5-6 年级、初中 9 年级

---

### 模式 5: 🌀 旋转体形成

**学习目标**: 理解平面图形旋转形成旋转体的概念

**玩法**:
- 显示一个平面图形（长方形、三角形、半圆等）和一条旋转轴
- 玩家预测旋转后形成的立体图形
- 答对后，播放旋转动画（2D→3D）

**典型对应**:
- 长方形绕一边旋转 → 圆柱
- 直角三角形绕直角边旋转 → 圆锥
- 半圆绕直径旋转 → 球体
- 梯形绕底边旋转 → 圆台

**对应课程**: 初中 9 年级、高中必修二

---

### 模式 6: 📍 空间位置关系

**学习目标**: 理解点、线、面之间的位置关系

**玩法**:
- 显示一个 3D 场景（如立方体框架）
- 标注某些点、线、面
- 问题如："这条线与这个面是什么关系？"
- 选项：平行、垂直、相交、在面内

**题目类型**:
- 线与线的关系（平行、相交、异面）
- 线与面的关系（平行、垂直、相交）
- 面与面的关系（平行、垂直、相交）

**对应课程**: 初中 7 年级、高中必修二

## 🎮 游戏系统设计

### 进度系统
```
玩家等级 = 所有模式总经验值
每个模式有独立的关卡进度
```

### 成就系统
| 成就 | 条件 |
|------|------|
| 视图大师 | 视图配对模式通关 20 关 |
| 展开专家 | 展开图模式连续答对 10 次 |
| 截面高手 | 截面模式获得 1000 分 |
| 计算达人 | 体积/表面积模式全对通关 |
| 空间想象王 | 全部模式达到黄金段位 |

### 奖励机制
- ⭐ 星星：每关根据准确率获得（1-3 星）
- 💎 宝石：连续答对奖励
- 🏆 奖杯：模式通关奖励
- 🎖️ 徽章：成就解锁

## 📱 技术实现方案

### 文件结构
```
solid-game/
├── index.html          # 主页面（模式选择入口）
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── main.js         # 主程序（模式选择、初始化）
│   ├── game.js         # 游戏核心逻辑
│   ├── shapes.js       # 几何体定义
│   ├── modes/
│   │   ├── view-match.js    # 模式 1: 视图配对
│   │   ├── net-match.js     # 模式 2: 展开图配对
│   │   ├── cross-section.js # 模式 3: 截面猜猜看
│   │   ├── volume-calc.js   # 模式 4: 体积/表面积
│   │   ├── rotation.js      # 模式 5: 旋转体
│   │   └── position.js      # 模式 6: 位置关系
│   └── utils/
│       ├── animations.js    # 动画效果
│       ├── sounds.js        # 音效
│       └── storage.js       # 本地存储
└── README.md
```

### 技术栈
- **Three.js r128**: 3D 渲染
- **原生 JavaScript ES6**: 游戏逻辑
- **CSS3**: UI 样式和动画
- **localStorage**: 进度保存
- **无构建工具**: 保持简单，直接浏览器运行

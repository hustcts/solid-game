# 🎮 Solid Game - 空间思维训练游戏

一款专为**小学生、初中生、高中生**设计的空间思维能力训练游戏，通过趣味互动的方式提升青少年的空间想象力和逻辑思维能力。

## 🌟 项目简介

**Solid Game** 是一款教育类游戏项目，旨在通过游戏化的方式帮助 K12 学生（6-18 岁）发展和锻炼空间思维能力。空间思维是 STEM 学习的重要基础，对于数学、物理、工程、计算机等学科的学习至关重要。

## 🎯 教育目标

### 核心能力培养
- **空间想象力** - 在脑海中构建和操作三维物体
- **空间可视化** - 理解和转换二维与三维表示
- **空间关系理解** - 把握物体之间的位置、方向和距离关系
- **心理旋转能力** - 在思维中旋转和操作物体

### 分年龄段目标

| 学段 | 年龄 | 训练重点 |
|------|------|----------|
| **小学** | 6-12 岁 | 基础形状认知、简单空间关系、方向感 |
| **初中** | 12-15 岁 | 三维转换、几何推理、空间分析 |
| **高中** | 15-18 岁 | 复杂空间问题、抽象思维、综合应用 |

## 🎲 游戏特色

- ✅ **循序渐进** - 难度分级，适合不同年龄段
- ✅ **趣味互动** - 游戏化设计，寓教于乐
- ✅ **即时反馈** - 实时评估和提示
- ✅ **科学设计** - 基于认知心理学和教育学研究
- ✅ **多平台支持** - 网页、移动端均可访问

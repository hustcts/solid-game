# 🎮 Solid Game 2.0 - 立体几何学习游戏

一款专为**小学生、初中生、高中生**设计的立体几何学习游戏，通过 6 种有趣的游戏模式，帮助学生深入理解立体几何概念！

## 🌟 6 大游戏模式

| 模式 | 图标 | 适合年级 | 学习目标 |
|------|------|----------|----------|
| **视图配对** | 👁️ | 小学 3 年级+ | 理解三视图（主视图、俯视图、左视图） |
| **展开图配对** | 📐 | 小学 4 年级+ | 掌握立体图形的平面展开 |
| **截面猜猜看** | 🔪 | 初中 8 年级+ | 理解平面截立体得到的截面形状 |
| **体积·表面积** | 🧮 | 小学 5 年级+ | 熟练计算几何体的体积和表面积 |
| **旋转体形成** | 🌀 | 初中 9 年级+ | 理解平面图形旋转形成旋转体 |
| **空间位置关系** | 📍 | 初中 7 年级+ | 判断点、线、面的位置关系 |

- **空间想象力** - 在脑海中构建和操作三维物体

- 高中：空间几何体综合应用

## 📖 游戏模式详解

### 模式 1：视图配对 👁️

**学习目标**：培养空间想象力，理解从不同方向观察物体的概念

**玩法**：
- 观察屏幕中央的 3D 几何体
- 从 4 组三视图中选择正确的一组
- 答对后进入下一关

**知识点**：
- 主视图（从正面看）
- 俯视图（从上面看）
- 左视图（从左侧看）

---

### 模式 2：展开图配对 📐

**学习目标**：理解立体图形与平面图形的转换关系

**玩法**：
- 观察 3D 几何体
- 选择正确的平面展开图
- 答对后播放展开动画

**覆盖几何体**：
- 正方体（11 种展开图）
- 长方体
- 圆柱体
- 圆锥体
- 三棱柱

---

### 模式 3：截面猜猜看 🔪

**学习目标**：理解平面截立体图形得到的截面形状

**玩法**：
- 观察几何体和切割平面
- 预测截面的形状
- 答对后展示切割动画

**典型题目**：
- 用平面截正方体，可能得到什么形状？
- 平行于底面截圆锥，得到什么形状？
- 斜着截圆柱，得到什么形状？

---

### 模式 4：体积·表面积 🧮

**学习目标**：掌握立体图形的体积和表面积公式

**玩法**：
- 观察标注尺寸的几何体
- 计算体积或表面积
- 从选项中选择正确答案

**公式覆盖**：
- 长方体：V = lwh, S = 2(lw + lh + wh)
- 正方体：V = a³, S = 6a²
- 圆柱：V = πr²h, S = 2πr² + 2πrh
- 圆锥：V = (1/3)πr²h
- 球体：V = (4/3)πr³, S = 4πr²

---

### 模式 5：旋转体形成 🌀

**学习目标**：理解平面图形旋转形成旋转体的概念

**玩法**：
- 观察平面图形和旋转轴
- 预测旋转后形成的立体图形
- 答对后播放旋转动画

**典型对应**：
- 长方形绕一边旋转 → 圆柱
- 直角三角形绕直角边旋转 → 圆锥
- 半圆绕直径旋转 → 球体
- 直角梯形绕垂直边旋转 → 圆台

---

### 模式 6：空间位置关系 📍

**学习目标**：理解点、线、面之间的位置关系

**玩法**：
- 观察 3D 场景中的几何元素
- 判断位置关系（平行、垂直、相交等）
- 选择正确答案

**题目类型**：
- 线与线的关系（平行、相交、异面）
- 线与面的关系（平行、垂直、相交）
- 面与面的关系（平行、垂直、相交）

## 💾 进度保存

游戏自动保存以下进度到本地：
- ⭐ 总星星数
- 💎 宝石数量
- 🏆 玩家等级
- 🎮 游戏次数
- 各模式通关进度

进度保存在浏览器本地存储（localStorage），清除浏览器数据会重置进度。
